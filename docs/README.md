# Documentation Sources

- `reports/`: canonical metric-focused study source.
- `companions/`: narrower companion extracts that point back to the canonical report.
- `reference/`: simulation model and code-path documentation.
- `archive/`: superseded source notes kept for traceability.

Generated figures, CSVs, manifests, rendered HTML, rendered PDF, and other
derived artifacts live under `../outputs/`.

Metric report artifacts are generated by `make figures` into
`../outputs/reports/metric_analysis/`.
